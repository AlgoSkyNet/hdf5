# HDF5

The package provides a reader and writer of [HDF5][1] files.

## [Documentation][doc]

## Installation

Fetch the package:

```bash
go get -d github.com/ready-steady/hdf5
```

Go to the directory of the package:

```bash
cd $GOPATH/src/github.com/ready-steady/hdf5
```

Install the package:

```bash
git submodule update --init
make install
```
